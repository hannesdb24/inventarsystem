# Inventarsystem – Starten

## Voraussetzungen

- [Node.js](https://nodejs.org) muss installiert sein (Version 18 oder neuer)

## Erste Einrichtung (einmalig)

Öffne ein Terminal im Ordner `Inventarsystem` und führe aus:

```
npm install
```

## System starten

```
npm start
```

Danach im Browser öffnen: **http://localhost:3000**

## Hinweise

- Die Datenbank wird automatisch als `inventar.db` im selben Ordner erstellt.
- Daten bleiben auch nach dem Beenden des Servers erhalten.
- Zum Beenden des Servers: `Strg + C` im Terminal.
